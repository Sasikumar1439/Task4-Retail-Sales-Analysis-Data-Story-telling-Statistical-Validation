# 📊 Retail Sales Data Analysis Project

## 🎯 Objective
To analyze retail transaction data and uncover insights into customer behavior, revenue trends, and business performance.

## 📌 Key Analysis
- Exploratory Data Analysis (EDA)
- Customer Segmentation
- Revenue Trend Analysis
- Payment Method Analysis

## 🧪 Hypothesis Testing
- Tested whether payment method affects customer spending
- Result: No significant impact (p-value = 0.79)

## 📈 Tools Used
- Python (Pandas, Matplotlib)
- SQL
- Power BI

## 📊 Deliverables
- Interactive Dashboard
- Business Presentation
- Data Storytelling Report

## 💡 Key Insights
- High-value customers drive revenue
- Payment method has no significant effect on spending
- Data-driven strategies improve decision-making
